UI/UX Design Prompt: Telegram Clone
Project Overview
Create a fully functional Telegram messenger UI/UX design clone. The design should be pixel-perfect, modern, and follow Material Design 3 principles with Telegram's signature aesthetic.

Primary Background: #FFFFFF (Light) / #0E0E0E (Dark)
Secondary Background: #F2F3F5 (Light) / #121212 (Dark)
Accent Color: #2481CC (Telegram Blue)
Secondary Accent: #40C351 (Online Green)
Destructive: #EF4444 (Red)
Text Primary: #1C1C1E (Light) / #FFFFFF (Dark)
Text Secondary: #8E8E93 (Gray)
Dividers: #E5E5EA (Light) / #2C2C2E (Dark)
Typography
Font Family: SF Pro Display (iOS), Roboto (Android), Inter (Web)
Heading 1: 24px, Bold, -0.5px
Heading 2: 20px, SemiBold
Body: 16px, Regular
Caption: 13px, Regular
Timestamp: 12px, Medium, #8E8E93
